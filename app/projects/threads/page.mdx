## Technology Stack and Features

- [**Laravel**](https://laravel.com) for the backend.
    - [Inertiajs](https://inertiajs.com) for single page app with [**Laravel**](https://laravel.com) and [**React**](https://react.dev).
    - [MySQL](https://www.mysql.org) as the SQL database.
- [React](https://react.dev) for the frontend.
    - Using hooks, Vite, and other parts of a modern frontend stack.
    - [Tailwindcss](https://tailwindcss.com) for the frontend components.
    - Build on dark mode.
